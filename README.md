# tic-tac-toe
This project is a Command-Line Interface (CLI) Tic Tac Toe game developed in Java. It allows two players to play against each other on a 3 X 3 grid.

# CLI Tic Tac Toe (Java)

A simple, interactive Tic Tac Toe game built using Java.

## Features
* **Two-Player Gameplay:** Alternates turns between 'X' and 'O'.
* **Dynamic Board Display:** Shows the current board state with indices after every move.
* **Input Validation:** Ensures players select valid, empty cells.
* **Win/Draw Detection:** Automatically announces the winner or a draw.
* **Replayability:** Allows players to restart a new game after completion.

## How to Play
1. [cite_start]The game displays a 3x3 grid with indices (0, 1, 2)[cite: 7, 13].
2. Players enter their desired row index and column index separately when prompted.
3. [cite_start]Align three of your symbols (horizontally, vertically, or diagonally) to win.

## How to Run
1. Ensure you have the Java Development Kit (JDK) installed.
2. Compile the file: `javac tic.java`
3. Run the game: `java tic`
